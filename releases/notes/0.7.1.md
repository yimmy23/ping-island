## 亮点
- `0.7.1` 聚焦 Island 在 MacBook 原生刘海屏和“更多空间”缩放分辨率下的显示稳定性，让主内容、用量提示和宠物图标更少被裁切或遮挡。
- 宠物动画渲染改为更克制的刷新节奏，降低空闲、工作和提醒状态下的 CPU/GPU 压力。

## 修复
- 修复 macOS “更多空间 (1800x1169)” 分辨率下 Island 宽度和内容边距不匹配，导致宠物、计数器或会话剩余提示被裁切的问题，对应 issue `#107`。
- 调整内置用量提示在刘海区域附近的展示方式，避免模型用量信息被屏幕原生黑色区域遮挡，对应 issue `#109`。
- 优化宠物动画的渲染刷新率和图层缓存，减少部分机器上的掉帧与额外耗电，对应 PR `#114`。

## 说明
- `0.7.1` 是 `0.7.0` 之后的补丁版本，主要覆盖显示缩放、物理刘海避让和动画性能。
- 本次发布不改变 hook、远程连接或更新通道语义。

- `#109` 模型用量被刘海遮挡

## Highlights
- `0.7.1` focuses on Island display stability around MacBook notch hardware and the macOS “More Space” scaled resolution, keeping core content, usage hints, and mascot chrome from clipping.
- Mascot animation rendering now uses a quieter refresh cadence to reduce CPU/GPU pressure while idle, active, or warning.

## Fixes
- Fixed Island width and content inset mismatches at macOS “More Space (1800x1169)” resolution that could clip the mascot, counters, or session-rest indicators, matching issue `#107`.
- Adjusted built-in usage hint presentation near the notch so model usage details are not hidden behind the physical screen cutout, matching issue `#109`.
- Optimized mascot animation refresh rates and layer caching to reduce frame drops and unnecessary battery use, matching PR `#114`.

## Notes
- `0.7.1` is the patch release after `0.7.0`, focused on display scaling, physical notch avoidance, and animation performance.
- This release does not change hook behavior, remote connection semantics, or the update channel.
