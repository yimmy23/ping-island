## 亮点
- Island 现在会根据当前聚焦的 App 或窗口迁移到对应屏幕，多显示器使用时更容易跟随正在工作的桌面。
- 启动时优先使用鼠标所在屏幕，减少外接显示器场景下默认落到内建屏幕或主屏幕的情况。

## 修复
- 修复同一 App 内切换窗口时 Island 可能不跟随目标屏幕的问题。
- 补充屏幕选择辅助逻辑，让按坐标定位屏幕和显式迁移窗口的行为更集中。

## 说明
- Sparkle build 号提升到 `51`，版本号提升到 `0.18.1`。
- 本次是基于 `main` 的小版本发布，聚焦多显示器与窗口焦点迁移体验。

## Highlights
- Island now migrates to the screen associated with the focused app or window, making multi-display setups follow the active workspace more naturally.
- Startup prefers the screen under the cursor, reducing cases where Island appears on the built-in or main display instead of the display in use.

## Fixes
- Fixed cases where switching between windows in the same app could leave Island on the previous screen.
- Added focused screen-selection helpers so coordinate-based screen lookup and explicit window migration stay centralized.

## Notes
- The Sparkle build number is now `51`, and the app version is now `0.18.1`.
- This is a small `main`-based release focused on multi-display and window-focus migration behavior.
