# Ping Island 0.4.0

## 亮点

- 让 detached floating pet 的悬停、点击和展开交互更贴近锚定式气泡，打开内容时更稳定也更有上下文感。
- 恢复首启欢迎流程，并补强浮动宠物模式的本地化 onboarding 与设置入口引导。
- 支持在更老的 glibc 环境下为远程 bridge 注入兼容库，同时补上静态 Linux 交叉编译的 musl 支持。
- Claude 和 Codex CLI 会话现在支持从 Island 直接继续 follow-up 输入，交互链路更完整。
- 会在空闲且浏览器全屏时自动隐藏 Island，减少对沉浸式工作流的打扰。

## 修复

- 修复 OpenClaw 被误识别成 Cursor 的问题，避免客户端归属和展示混乱。
- 修复 SSH 隧道因系统睡眠中断后，远程 agent 重连不稳定的问题。
- 稳定 detached island 的拖拽、交互接力和重新停靠表现。
- 修复设置页 tab 切换后的滚动位置串页问题，并提升设置持久化相关 CI 稳定性。
- 让 Codex 详情导航、设置文案和远程状态说明更完整地跟随应用语言切换。

## 说明

- 这一版是一次覆盖交互体验、远程兼容性和多语言细节的综合更新，适合直接作为 `0.4.0` 里程碑发布。
