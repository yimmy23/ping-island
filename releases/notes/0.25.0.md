## Compatibility
- Adds compatibility with the new Hooks flow in QoderWork `0.6.7+`
- QoderWork ordinary tool permission notifications remain non-blocking and do not show Island approval prompts
- QoderWork `AskUserQuestion` still appears as an external-handling prompt, avoiding responses on non-responsive Hooks paths

## Notes
- The Sparkle build number is now `64`, and the app version is now `0.25.0`
- This release is based on latest `main` and is published as GitHub latest
