## 亮点
- 恢复 tmux 内 CLI 会话的 Island 聊天回复：Claude Code、Codex CLI 和 Qoder CLI 的终端会话现在可以从 Island 聊天框把文本可靠发送回对应 tmux pane。

## Highlights
- Restored Island chat replies for CLI sessions running inside tmux. Claude Code, Codex CLI, and Qoder CLI terminal sessions can now receive text from the Island chat box through the matching tmux pane.
- Made the empty hover panel lighter and localized. When no active sessions are available, hover, click, and shortcut hints now fit the notch panel without the larger session-list footprint.
- Made startup quieter. Ping Island now prepares the docked or floating surface at launch without stealing focus or expanding stale session content.

## Fixes
- Fixed tmux-hosted Codex and compatible CLI sessions taking the wrong app-server continuation path instead of sending Island chat input back to the terminal.
- Fixed previously queued notification state opening expanded content immediately during app startup; only user actions or fresh post-launch notifications should open it.
- Fixed empty hover guidance copy and initial panel sizing.

## Notes
- The Sparkle build number is now `30`, and the app version is now `0.9.0`.
