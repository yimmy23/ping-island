- 升级到 `0.5.3` 后，停靠在刘海区的 Ping Island 界面会移除顶部标题拖拽 affordance，让默认停靠态更干净、更聚焦。

## Highlights
- Upgrading to `0.5.3` removes the header drag affordance from the docked Ping Island UI, making the default docked state cleaner and more focused.
- This release is cut directly from the latest `main` branch and includes the newest resource package update so shipped assets stay aligned with the current app behavior.

## Fixes
- Removed the extra drag hint from the docked Island header to reduce visual noise and keep closed/open states from competing for attention.
- Synced the latest resource package update so the release matches the current mainline repository content.

## Notes
- `0.5.3` is a patch release for the current `0.5.x` line, and existing users should upgrade in place.
- This release includes the UI cleanup and resource refresh merged into `main` after `v0.5.2`, without expanding the feature scope.
