## Highlights
- Upgrading to `0.5.2` restores the version-matched upgrade onboarding for returning notched-display users, so the `0.5.1` guidance is no longer skipped because of stale local state.
- This release is cut directly from the latest `main` branch and focuses on closing the last upgrade-path UX gap left after `0.5.1`.

## Fixes
- Fixed a case where returning notched-display users who upgraded to `0.5.1` would not receive the off-island Buddy / display-mode upgrade guidance again.
- Aligned the version checks in app launch configuration and the hook installer so one-time onboarding flows advance with the actually shipped client version.

## Notes
- `0.5.2` is a patch release for `0.5.1`, and all current `0.5.x` users should upgrade in place.
- This release contains only the upgrade-onboarding recovery fix merged into `main` after `v0.5.1`, with no additional feature scope.
