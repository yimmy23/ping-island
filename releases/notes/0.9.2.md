## 修复
- 修复 MacBook 原生刘海屏在调整显示分辨率或缩放后，Island 继续使用旧屏幕几何，导致左侧图标可能被真实刘海遮挡的问题。
- Island 现在会在屏幕参数变化时刷新闭合宽度、命中区域和展开面板限制；物理刘海兜底宽度保持稳定，不会随显示分辨率变宽而漂移。

## 说明
- Sparkle build 号提升到 `32`，版本号提升到 `0.9.2`。
- 关联问题：#135。

<!-- en -->
## Fixes
- Fixed a notched MacBook display issue where changing display resolution or scaling could leave Island using stale screen geometry, allowing the left icon to sit under the physical camera housing.
- Island now refreshes closed width, hit regions, and opened-panel limits when screen parameters change, while keeping the physical-notch fallback width stable instead of scaling it with display resolution.

## Notes
- The Sparkle build number is now `32`, and the app version is now `0.9.2`.
- Related issue: #135.
