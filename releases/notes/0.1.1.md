## Highlights
- Adds WorkBuddy hook and session recognition so CodeBuddy and WorkBuddy monitoring coverage stays in sync.
- Introduces a native Claude/Codex runtime preview path to reduce reliance on hook-only monitoring over time.
- Strengthens the SSH remote workflow with remote OpenClaw hook bootstrap and clearer remote access surfacing.

## Fixes
- The closed-notch badge now counts only actionable sessions instead of every inactive row.
- Stabilizes the native Claude launch path and tightens session state and list presentation details.

## Notes
- `0.1.1` is a focused client-monitoring quality release with remote workflow polish.
