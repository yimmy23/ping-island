## Ping Island 0.0.3

### Highlights
- Adds a Remote Hosts management panel inside Settings for SSH-based bridge setup.
- Introduces Linux-capable remote bridge delivery through GitHub Releases for supported SSH hosts.
- Improves remote install diagnostics and localizes the new remote-host management copy.

### Notes
- Linux remote-host bootstrap currently ships a `PingIslandBridge-linux-x86_64` asset.
- The release workflow now publishes signed macOS app packages and the Linux remote bridge artifact together.
