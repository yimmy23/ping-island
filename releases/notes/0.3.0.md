## 亮点
- 新增 Codex 上下文压缩通知，压缩完成后会通过 Island 提示，同时支持独立开关控制。
- 正式支持 Codex、Qoder 等会话里的子 Agent 会话识别与展示，列表、悬浮预览和状态流都能看见更细的执行层级。
- 优化 OpenClaw 宠物形象，让小龙虾特征在菜单栏和 Island 小尺寸场景下更清晰也更可爱。
- 优化 SSH Remote 资源包体积与下发方式，远程桥接安装更轻，监控链路更顺滑。

## 修复
- 修复 SSH 远程场景里重复审批请求会阻塞会话的问题，远程工作流更稳定。
- 修复全屏偏好与系统刘海尺寸下的 Island 显示细节，避免不必要的收起和布局误差。

## 说明
- `0.3.0` 是一次聚焦 Codex/Qoder 可见性、远程监控体验和客户端形象打磨的功能更新。

## Highlights
- Adds Codex context-compaction notifications with an independent toggle, so Island can remind you when compression finishes without affecting other completion popups.
- Officially supports subagent-session recognition and presentation for Codex, Qoder, and similar session trees across the list, hover preview, and state flow.
- Refreshes the OpenClaw mascot so the crayfish silhouette reads more clearly and more playfully at menu-bar and Island scale.
- Shrinks and repackages SSH Remote bridge assets so remote bootstrap and monitoring feel lighter and smoother.

## Fixes
- Fixes duplicate remote approval requests that could block SSH sessions.
- Tightens fullscreen-preference and system-notch sizing behavior so Island visibility stays more predictable.

## Notes
- `0.3.0` is a feature release centered on Codex/Qoder visibility, remote-monitoring polish, and client-branding refinement.
