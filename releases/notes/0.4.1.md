# Ping Island 0.4.1

## 亮点

- 让 detached floating pet 的反馈更连贯，完成态提示与浮动宠物的表现现在保持一致。
- 首次使用 detached 模式时会给出一次性的拖拽引导，帮助新用户更快理解“宠物离岛工作”的交互。
- 拖拽宠物时新增专用动作表现，拖动反馈更明确。

## 修复

- 修复 detached 浮动气泡的一处主线程布局问题，避免 `0.4.0` 的 release 打包链路不稳定。
- 稳定 detached 模式首启提示、悬停和拖拽之间的状态切换。

## 说明

- `0.4.1` 是一次以 detached 宠物交互和发布稳定性为主的小版本更新，建议直接覆盖安装 `0.4.0`。
