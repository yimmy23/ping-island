## 亮点
- 继续打磨自动更新与在线 release notes 体验，更新说明窗口、庆祝反馈和静默更新策略都更稳定顺手。
- 设置窗口现在会在重新打开时恢复显示，并为详情面板提供更合适的默认宽度。

## 说明
- 这一版主要整合 `0.0.8` 之后的更新体验、窗口细节与交互反馈改进，为后续正式自动更新发布做准备。

## Highlights
- Continued polishing the auto-update and online release-notes flow so the update window, celebration feedback, and silent-update policy behave more reliably.
- The settings window now restores itself when reopened and uses a more comfortable default width for the detail pane.
- Added more shortcut entry points across the Island and release-notes UI so frequent view switching feels faster without interrupting active sessions.

## Notes
- This release packages the user-facing update UX, window-behavior, and interaction refinements that landed after `0.0.8`, setting up the next formal auto-update release.
