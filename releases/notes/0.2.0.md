## 亮点
- 新增全局快捷键支持，可直接打开当前活跃会话或会话列表，减少在菜单栏和终端之间来回切换。
- 正式支持 SSH 远程设备监控，可通过 SSH 引导远程 `PingIslandBridge`，并把远程事件回流到本机 Island 界面。
- 新增 Hermes Agent 官方 plugin hooks 接入，现可通过 `~/.hermes/plugins/ping_island/` 统一接入会话事件。
- 新增 Qwen Code 官方 hooks 接入，现可通过 `~/.qwen/settings.json` 追踪通知、追问与会话状态。

## 说明
- `0.2.0` 是一次以跨终端工作流和多客户端覆盖扩展为重点的功能版本。

<!-- en -->
## Highlights
- Adds global shortcuts so you can jump straight to the active session or the full session list without bouncing between the menu bar and terminal windows.
- Ships formal SSH remote monitoring support by bootstrapping `PingIslandBridge` over SSH and forwarding remote events back into the local Island UI.
- Adds Hermes Agent support through the official plugin-hooks integration at `~/.hermes/plugins/ping_island/`.
- Adds Qwen Code support through the official hooks integration at `~/.qwen/settings.json`, including notifications, follow-up prompts, and session state tracking.

## Notes
- `0.2.0` is a feature-focused release centered on cross-terminal workflows and broader client coverage.
