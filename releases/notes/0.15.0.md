## Highlights
- Added the anonymous usage telemetry switch and first-run consent checkbox; telemetry only uploads after user consent.
- Extended telemetry to Island open/close interactions plus notification and approval request/resolution buckets, so the interaction flow can be improved without collecting content.
- Added idle-user protection so blocking approvals and questions can route back to the terminal when the user is away instead of stalling agents.

## Fixes
- Fixed Codex work that was still running but did not appear or notify reliably in Ping Island.
- Fixed hook bridge IPC and hook-directory authorization writes in App Store sandbox builds.
- Continued improving Gemini hook-family detection and per-event 8-bit sound configuration to reduce client misclassification.

## Notes
- The Sparkle build number is now `44`, and the app version is now `0.15.0`.
- Anonymous telemetry still uses allowlisted fields only and does not collect prompts, responses, paths, project names, hostnames, SSH targets, or session IDs.
