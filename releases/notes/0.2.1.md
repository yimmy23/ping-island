## 亮点
- 优化 Qwen Code hooks 兼容性，补上 `ask_user_question` 在多次 hook 交接时的答案保留与回流，减少追问流程丢失上下文的问题。
- 调整 Island 刘海屏尺寸策略，现会根据不同 Mac 机型的真实刘海宽高自动匹配闭合态高度与宽度，减少多设备上的遮挡和裁切。

## 修复
- 修复部分 Qwen 问答选项在 UI 中被裁切的问题，让较长选项也能完整显示。
- 修复更宽物理刘海机型上的闭合态 Island 估算偏窄问题。

## 说明
- `0.2.1` 是一次以兼容性和设备适配为重点的稳定性版本，主要聚焦 Qwen Code 工作流与多机型刘海屏表现。

## Highlights
- Improves Qwen Code hook compatibility so `ask_user_question` answers survive multi-hook handoffs instead of dropping context mid-follow-up.
- Adapts the Island to each Mac's physical notch dimensions, matching the closed-state width and height more accurately across multiple devices.

## Fixes
- Fixes clipped Qwen follow-up options in the UI so longer answers remain fully readable.
- Fixes closed Island sizing on Macs with wider physical notches.

## Notes
- `0.2.1` is a stability-focused release centered on Qwen Code compatibility and multi-device notch sizing.
