## Fixes
- Fixed visible idle pets being frozen into static frames by the low-power policy, so the floating pet and Island pet keep visible animation again.
- Raised the mascot Canvas render cadence so idle, working, warning, and dragging states read as continuous motion again.
- Fixed the two bottom Top 5 cards in Settings analytics collapsing into separate rows; Agent type ranking and Tool Calls Top 5 now stay in a stable 1:1 row.

## Notes
- The Sparkle build number is now `53`, and the app version is now `0.19.1`.
- This latest-`main` patch release is scoped to mascot animation visibility and analytics layout fixes on top of `0.19.0`.
